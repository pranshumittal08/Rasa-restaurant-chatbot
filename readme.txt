Objective

To create a simple chatbot using Rasa framework that would take in the preferences from a user about the cuisine, city (Tier 1 and Tier 2 only) and budget and then
provide recommendations of top choices sorted according to the ratings. The project uses Zomato API to fetch the restaurants. The bot is also able to send 
recommendations over email if allowed.

Versions of major packages used:

python version - 3.7.9
tensorflow version - 2.3.1
rasa version - 2.0
